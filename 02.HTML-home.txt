블럭요소(block element)와 인라인요소(inline element)란?

	블럭요소 : 하나의 태그가 브라우저에서 좌우공간을 다 차지하면서 독립적인 덩어리 공간을 가지는 요소들
		: <div>,<table>,<h1>~<h6>,<p>,<form>,<ul>,<ol>,<li>,<dl>,<dt>,<dd>,<pre>,<blockquote>등
		: 너비, 높이 지정 가능, 외부여백(margin) 상하 좌우 적용가능
		
	인라인요소 
		: 하나의 태그가 브라우저에서 실제로 코딩된 그 영역만 차지하여 좌우로 다른 태그가 나란히 위치할 수 있는 요소들
		: 행 안의 일부분, 텍스트레벨 요소라고도 함
		: <span>,<a>,<br>,<em>,<strong>,<input>,<label>,<img>
		: 너비, 높이 지정못함, 외부여백(margin)  좌우만 적용가능

미디어 쿼리

1. 미디어 쿼리란?
	-  사이트에 접속하는 장치에 따라 특정한 CSS 스타일을 사용하도록 해 줍니다

2. 미디어 쿼리 구문

@media [only | not] 미디어 유형 [and 조건] * [and 조건]

- 너비가 200px이상이고 360px 이하일때 적용되는 스타일
@media screen and (min-width: 200px) and (max-width: 360px) {

}

1) 미디어 유형의 종류
	- all : 모든 미디어 유형
	- print : 인쇄 장치
	- screen : 컴퓨터 스크린(스마트폰 스크린 포함)

2) 미디어 쿼리의 조건
	- width, height
	- min-width, min-height
	- max-width, max-height
	- orientation: 장비방향
		portrait :세로 모드
		landscape :가로 모드
	
	- 뷰포트의 너비가 600px 이상이고 959px 이하일 때
	@media all (min-width: 600px) and (max-width: 959px) {
    ...
	}
	

        <style>
            p {
                font-size: 13px;
            }
            
            @media screen and (min-width: 300px) { /* 300px이상일때 */
                body {
                    background: orange;
                }
            }
            @media screen and (min-width: 600px) and (max-width:800px){ /* 600px이상 1000px 이하*/
                body {
                    background: blue;
                }
            }
            
            @media screen and (orientation: landscape) { /*가로모드일때*/
                body {
                    background: gray;
                }
            }
        </style>
	
3. 미디어 쿼리 적용하기
1) <link> 태그 사용하기
- <link rel="stylesheet" media="미디어 쿼리 조건" href="css 파일 경로">
	
	<head>
	<link rel="stylesheet" type="text/css" href="css/style.css" media="screen and (min-width:600px) and (max-width: 800px)">
	</head>
	
- <link rel="stylesheet" media="screen and (max-width: 768px)" href="css/table.css">




2) @import 구문 사용하기
	- @import url(css 파일 경로) 미디어 쿼리 조건
	- @import url("css/table.css") screen and (min-width: 321px) and (max-width: 768px);
