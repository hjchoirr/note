4. 스타일과 스타일 시트
	1) 내부 스타일 시트 
		- 웹 문서 안에서 사용할 스타일을 문서 안에 정의한 것
		  <style> 스타일정의 </style>
		
	2) 외부 스타일 시트
		- <link rel="stylesheet" type="text/css" href="외부 스타일 파일 경로">
		- 확장자 : .css
		
	3) 인라인 스타일
		태그속성 - style 속성에 직접 스타일 정의
